1.1- ¿qué es una URL?

URL es una dirección a un único recurso en la web, como páginas HTML, documentos CSS, o imagenes. Es común apuntar un recurso no existido o movido, dependiendo del servidor web y su URL manejada adecuadamente.

1.2- ¿qué es una url absoluta?

Una URL absoluta es una dirección web completa que incluye el protocolo, el nombre de dominio y la ruta completa del archivo o recurso.

1.3- ¿qué es una url relativa?

Las URL relativas son estructuras de URL que contienen la ruta de ubicación, omitiendo el nombre del sitio web

1.4- ¿cuál es la diferencia entre una url relativa y una url absoluta?

Una URL absoluta contiene toda la información necesaria para localizar un recurso, mientras que una URL relativa utiliza una URL absoluta como punto de partida, formando la "URL completa" del destino.

1.10- ¿Cuál es el propósito de la etiqueta anchor?

El Elemento HTML Anchor crea un enlace a otras páginas de internet, archivos, ubicaciones, direcciones de correo o cualquier URL.

1.11- ¿Qué atributo se utiliza en la etiqueta anchor para especificar la URL a la que debe enlazar?

El atributo que se utiliza es el href de un enlace es una etiqueta anchor que indica la relación entre páginas y los motores de búsqueda, consistiendo de un URL y un texto o objeto clicable que los usuarios verán en la página.

1.12- ¿Qué sucede si el atributo "href" en la etiqueta anchor está vacío?

lo que pasa es que la etiqueta a (anchor) no sirve sin el atributo y puede ser usado para crear un enlace muerto en una tarea donde no desea un enlace en vivo o sabe el destino.

1.13- ¿Cuál es el uso del atributo "target" en la etiqueta anchor y cuáles son los posibles valores que puede tener?

Especifica en donde desplegar la URL enlazada. los posibles valores son: _self - _blank - _parent - _top y _self
